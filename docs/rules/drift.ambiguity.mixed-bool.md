# `drift.ambiguity.mixed-bool`

| field    | value |
|----------|-------|
| category | `ambiguity` |
| default  | `info` |
| fixable  | `no` |

## what

pick one: TRUE / true. don't mix.

## bad

```sql
SELECT * FROM users WHERE active = 1 OR verified = TRUE;
```

## good

```sql
SELECT * FROM users WHERE active = TRUE AND verified = TRUE;

```
