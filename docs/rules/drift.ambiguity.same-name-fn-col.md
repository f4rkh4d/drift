# `drift.ambiguity.same-name-fn-col`

| field    | value |
|----------|-------|
| category | `ambiguity` |
| default  | `info` |
| fixable  | `no` |

## what

naming a column `count` or `current_date` invites parse ambiguity

## bad

```sql
SELECT count FROM page_views;
```

## good

```sql
SELECT view_count FROM page_views;

```
