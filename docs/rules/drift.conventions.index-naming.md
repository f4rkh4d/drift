# `drift.conventions.index-naming`

| field    | value |
|----------|-------|
| category | `conventions` |
| default  | `info` |
| fixable  | `no` |

## what

index names should be `ix_<table>_<cols>`

## bad

```sql
CREATE INDEX idx1 ON users (email);
```

## good

```sql
CREATE INDEX users_email_idx ON users (email);

```
