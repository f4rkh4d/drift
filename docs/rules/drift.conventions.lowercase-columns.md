# `drift.conventions.lowercase-columns`

| field    | value |
|----------|-------|
| category | `conventions` |
| default  | `info` |
| fixable  | `no` |

## what

column names should be lowercase snake_case

## bad

```sql
CREATE TABLE users (Id int, FullName text);
```

## good

```sql
CREATE TABLE users (id int, full_name text);

```
