# `drift.conventions.no-hungarian`

| field    | value |
|----------|-------|
| category | `conventions` |
| default  | `info` |
| fixable  | `no` |

## what

don't prefix columns with type (e.g. `str_name`, `int_count`)

## bad

```sql
CREATE TABLE users (
  intId int,
  strName text,
  boolActive boolean
);
```

## good

```sql
CREATE TABLE users (
  id int,
  name text,
  active boolean
);

```
