# `drift.conventions.pk-column-id`

| field    | value |
|----------|-------|
| category | `conventions` |
| default  | `info` |
| fixable  | `no` |

## what

primary key column should be `id` (not `user_id` inside users table)

## bad

```sql
CREATE TABLE users (
  user_id int PRIMARY KEY
);
```

## good

```sql
CREATE TABLE users (
  id int PRIMARY KEY
);

```
