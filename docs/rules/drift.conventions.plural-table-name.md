# `drift.conventions.plural-table-name`

| field    | value |
|----------|-------|
| category | `conventions` |
| default  | `info` |
| fixable  | `no` |

## what

table names should be plural (e.g. users, not user)

## bad

```sql
CREATE TABLE user (id int);
```

## good

```sql
CREATE TABLE users (id int);

```
