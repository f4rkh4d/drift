# `drift.correctness.cartesian-join`

| field    | value |
|----------|-------|
| category | `correctness` |
| default  | `warning` |
| fixable  | `no` |

## what

multiple tables in FROM with no WHERE or JOIN predicate

## bad

```sql
SELECT u.name, o.total
FROM users u, orders o;
```

## good

```sql
SELECT u.name, o.total
FROM users u
JOIN orders o ON o.user_id = u.id;

```
