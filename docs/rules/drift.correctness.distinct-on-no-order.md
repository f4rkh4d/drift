# `drift.correctness.distinct-on-no-order`

| field    | value |
|----------|-------|
| category | `correctness` |
| default  | `warning` |
| fixable  | `no` |

## what

DISTINCT ON without a matching ORDER BY returns arbitrary rows

## bad

```sql
SELECT DISTINCT ON (user_id) *
FROM events;
```

## good

```sql
SELECT DISTINCT ON (user_id) *
FROM events
ORDER BY user_id, created_at DESC;

```
