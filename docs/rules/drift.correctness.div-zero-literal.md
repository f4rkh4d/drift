# `drift.correctness.div-zero-literal`

| field    | value |
|----------|-------|
| category | `correctness` |
| default  | `error` |
| fixable  | `no` |

## what

`/ 0` as a literal is a guaranteed runtime error

## bad

```sql
SELECT total / 0 FROM invoices;
```

## good

```sql
SELECT CASE WHEN tax_rate = 0 THEN 0 ELSE total / tax_rate END
FROM invoices;

```
