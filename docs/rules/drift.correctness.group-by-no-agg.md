# `drift.correctness.group-by-no-agg`

| field    | value |
|----------|-------|
| category | `correctness` |
| default  | `info` |
| fixable  | `no` |

## what

GROUP BY with no aggregate is equivalent to DISTINCT

## bad

```sql
SELECT user_id
FROM events
GROUP BY user_id;
```

## good

```sql
SELECT DISTINCT user_id FROM events;

```
