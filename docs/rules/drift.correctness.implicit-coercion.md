# `drift.correctness.implicit-coercion`

| field    | value |
|----------|-------|
| category | `correctness` |
| default  | `warning` |
| fixable  | `no` |

## what

comparing a number column to a string literal ('5' vs 5) forces a coercion

## bad

```sql
SELECT * FROM orders WHERE total = '100';
```

## good

```sql
SELECT * FROM orders WHERE total = 100;

```
