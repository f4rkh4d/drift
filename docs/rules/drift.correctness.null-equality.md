# `drift.correctness.null-equality`

| field    | value |
|----------|-------|
| category | `correctness` |
| default  | `error` |
| fixable  | `no` |

## what

`x = NULL` is always unknown. use `x IS NULL`.

## bad

```sql
SELECT * FROM users WHERE deleted_at = NULL;
```

## good

```sql
SELECT * FROM users WHERE deleted_at IS NULL;

```
