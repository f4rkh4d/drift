# `drift.correctness.reserved-fn-name`

| field    | value |
|----------|-------|
| category | `correctness` |
| default  | `warning` |
| fixable  | `no` |

## what

CREATE FUNCTION with a reserved name will shadow built-ins

## bad

```sql
CREATE FUNCTION count(text) RETURNS int AS $$
  SELECT length($1)
$$ LANGUAGE sql;
```

## good

```sql
CREATE FUNCTION text_length(text) RETURNS int AS $$
  SELECT length($1)
$$ LANGUAGE sql;

```
