# `drift.correctness.union-vs-union-all`

| field    | value |
|----------|-------|
| category | `correctness` |
| default  | `info` |
| fixable  | `no` |

## what

plain UNION deduplicates, which you rarely want. be explicit.

## bad

```sql
SELECT id FROM customers
UNION
SELECT id FROM partners;
```

## good

```sql
SELECT id FROM customers
UNION ALL
SELECT id FROM partners;

```
