# `drift.performance.like-leading-wildcard`

| field    | value |
|----------|-------|
| category | `performance` |
| default  | `warning` |
| fixable  | `no` |

## what

LIKE '%foo' can't use a btree index. full table scan territory.

## bad

```sql
SELECT * FROM products WHERE name LIKE '%phone';
```

## good

```sql
SELECT * FROM products WHERE name LIKE 'phone%';

```
