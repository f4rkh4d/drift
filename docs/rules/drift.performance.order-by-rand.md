# `drift.performance.order-by-rand`

| field    | value |
|----------|-------|
| category | `performance` |
| default  | `warning` |
| fixable  | `no` |

## what

ORDER BY random() / RAND() sorts the whole table to pick N rows

## bad

```sql
SELECT * FROM users ORDER BY RANDOM() LIMIT 10;
```

## good

```sql
-- pick a stable random window:
SELECT * FROM users TABLESAMPLE SYSTEM (1) LIMIT 10;

```
