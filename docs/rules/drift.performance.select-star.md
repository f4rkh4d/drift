# `drift.performance.select-star`

| field    | value |
|----------|-------|
| category | `performance` |
| default  | `warning` |
| fixable  | `no` |

## what

SELECT * fetches columns you don't need and breaks when the schema changes

## bad

```sql
SELECT * FROM users;
```

## good

```sql
SELECT id, email, created_at FROM users;

```
