# `drift.portability.backtick-quote`

| field    | value |
|----------|-------|
| category | `portability` |
| default  | `warning` |
| fixable  | `no` |

## what

backticks are mysql/bigquery-only; ansi uses double quotes

## bad

```sql
SELECT `id`, `name` FROM `users`;
```

## good

```sql
SELECT "id", "name" FROM "users";

```
