# `drift.portability.dialect-fn`

| field    | value |
|----------|-------|
| category | `portability` |
| default  | `info` |
| fixable  | `no` |

## what

functions like GENERATE_SERIES, IFNULL, IF() are dialect-bound

## bad

```sql
SELECT NVL(name, 'unknown') FROM users;  -- oracle
```

## good

```sql
SELECT COALESCE(name, 'unknown') FROM users;

```
