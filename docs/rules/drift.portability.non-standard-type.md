# `drift.portability.non-standard-type`

| field    | value |
|----------|-------|
| category | `portability` |
| default  | `info` |
| fixable  | `no` |

## what

types like SERIAL, DATETIME, TINYINT are dialect-specific

## bad

```sql
CREATE TABLE t (data JSONB);  -- pg-only
```

## good

```sql
CREATE TABLE t (data JSON);  -- portable; pg can still index it

```
