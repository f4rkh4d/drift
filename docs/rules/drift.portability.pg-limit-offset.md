# `drift.portability.pg-limit-offset`

| field    | value |
|----------|-------|
| category | `portability` |
| default  | `info` |
| fixable  | `no` |

## what

LIMIT/OFFSET is postgres/mysql; ansi uses FETCH FIRST n ROWS ONLY

## bad

```sql
SELECT * FROM events LIMIT 10 OFFSET 100;
```

## good

```sql
-- ANSI form, works on more dialects:
SELECT * FROM events OFFSET 100 ROWS FETCH FIRST 10 ROWS ONLY;

```
