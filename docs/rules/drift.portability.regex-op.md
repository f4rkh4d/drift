# `drift.portability.regex-op`

| field    | value |
|----------|-------|
| category | `portability` |
| default  | `info` |
| fixable  | `no` |

## what

~ and ~* are postgres-only; mysql uses REGEXP

## bad

```sql
SELECT * FROM users WHERE email ~ '^foo';  -- pg-only
```

## good

```sql
SELECT * FROM users WHERE email LIKE 'foo%';
-- or, if you need real regex, wrap in REGEXP_LIKE etc.

```
