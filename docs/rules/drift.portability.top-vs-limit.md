# `drift.portability.top-vs-limit`

| field    | value |
|----------|-------|
| category | `portability` |
| default  | `warning` |
| fixable  | `no` |

## what

SELECT TOP is tsql-only; drift doesn't support tsql yet

## bad

```sql
SELECT TOP 10 * FROM users;
```

## good

```sql
SELECT * FROM users LIMIT 10;

```
