# `drift.security.grant-all`

| field    | value |
|----------|-------|
| category | `security` |
| default  | `warning` |
| fixable  | `no` |

## what

GRANT ALL is almost never what you want. specify the privileges.

## bad

```sql
GRANT ALL ON users TO app_user;
```

## good

```sql
GRANT SELECT, INSERT, UPDATE ON users TO app_user;

```
