# `drift.security.plaintext-password`

| field    | value |
|----------|-------|
| category | `security` |
| default  | `error` |
| fixable  | `no` |

## what

password literals in migrations leak to logs, backups, git history

## bad

```sql
CREATE TABLE users (
  id int PRIMARY KEY,
  password text
);
```

## good

```sql
CREATE TABLE users (
  id int PRIMARY KEY,
  password_hash text NOT NULL  -- bcrypt/argon2
);

```
