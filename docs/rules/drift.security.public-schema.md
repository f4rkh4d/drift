# `drift.security.public-schema`

| field    | value |
|----------|-------|
| category | `security` |
| default  | `info` |
| fixable  | `no` |

## what

public schema writes are an old postgres habit; scoped schemas are safer

## bad

```sql
CREATE TABLE public.tokens (id int, secret text);
```

## good

```sql
CREATE SCHEMA app;
CREATE TABLE app.tokens (id int, secret text);

```
