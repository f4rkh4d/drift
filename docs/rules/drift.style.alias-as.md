# `drift.style.alias-as`

| field    | value |
|----------|-------|
| category | `style` |
| default  | `info` |
| fixable  | `no` |

## what

use explicit AS for column aliases (select a AS b, not select a b)

## bad

```sql
SELECT u.id user_id FROM users u;
```

## good

```sql
SELECT u.id AS user_id FROM users AS u;

```
