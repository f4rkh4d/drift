# `drift.style.crlf`

| field    | value |
|----------|-------|
| category | `style` |
| default  | `info` |

## what

files should use LF line endings, not CRLF

## bad

```sql
SELECT id
FROM users;
```

## good

```sql
SELECT id
FROM users;

```
