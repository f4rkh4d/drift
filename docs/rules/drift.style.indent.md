# `drift.style.indent`

| field    | value |
|----------|-------|
| category | `style` |
| default  | `info` |
| fixable  | `no` |

## what

indentation should be a multiple of the configured indent width

## bad

```sql
SELECT id,
  name,
     email
FROM users;
```

## good

```sql
SELECT id,
       name,
       email
FROM users;

```
