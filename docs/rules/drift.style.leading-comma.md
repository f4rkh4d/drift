# `drift.style.leading-comma`

| field    | value |
|----------|-------|
| category | `style` |
| default  | `info` |
| fixable  | `no` |

## what

flags lines that start with a comma when the style is trailing

## bad

```sql
SELECT id
     , name
     , email
FROM users;
```

## good

```sql
SELECT id,
       name,
       email
FROM users;

```
