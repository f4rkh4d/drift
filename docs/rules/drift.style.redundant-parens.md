# drift.style.redundant-parens

**redundant parentheses**  —  category: `style`  severity: `info`  fixable: false

run `drift explain drift.style.redundant-parens` for the interactive version.

## why

see the rule description in `drift explain`. short version: this is a "style" rule, fires at `info` by default.

## configuration

```toml
[rules]
"drift.style.redundant-parens" = "error"     # or warning, info, off
```
