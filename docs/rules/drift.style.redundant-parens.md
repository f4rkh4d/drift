# `drift.style.redundant-parens`

| field    | value |
|----------|-------|
| category | `style` |
| default  | `info` |
| fixable  | `no` |

## what

flags `((expr))` — one level is enough

## bad

```sql
SELECT * FROM users WHERE ((id = 1));
```

## good

```sql
SELECT * FROM users WHERE id = 1;

```
