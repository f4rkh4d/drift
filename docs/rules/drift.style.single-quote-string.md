# `drift.style.single-quote-string`

| field    | value |
|----------|-------|
| category | `style` |
| default  | `warning` |
| fixable  | `no` |

## what

string literals should use single quotes (double quotes are identifiers in ansi sql)

## bad

```sql
SELECT * FROM users WHERE name = "farkhad";
```

## good

```sql
SELECT * FROM users WHERE name = 'farkhad';

```
