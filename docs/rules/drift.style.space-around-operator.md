# `drift.style.space-around-operator`

| field    | value |
|----------|-------|
| category | `style` |
| default  | `info` |
| fixable  | `no` |

## what

binary operators should have whitespace on both sides

## bad

```sql
SELECT * FROM users WHERE active=true AND id>10;
```

## good

```sql
SELECT * FROM users WHERE active = true AND id > 10;

```
