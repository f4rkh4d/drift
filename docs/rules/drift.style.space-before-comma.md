# `drift.style.space-before-comma`

| field    | value |
|----------|-------|
| category | `style` |
| default  | `info` |
| fixable  | `no` |

## what

do not put whitespace before commas

## bad

```sql
SELECT id ,name ,email FROM users;
```

## good

```sql
SELECT id, name, email FROM users;

```
