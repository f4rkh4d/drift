# drift.style.tab-indent

**tab indent** .  category: `style`  severity: `info`  fixable: false

run `drift explain drift.style.tab-indent` for the interactive version.

## why

see the rule description in `drift explain`. short version: this is a "style" rule, fires at `info` by default.

## configuration

```toml
[rules]
"drift.style.tab-indent" = "error"     # or warning, info, off
```
